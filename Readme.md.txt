# Titanic Passenger Survival Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard that analyzes the Titanic passenger dataset to identify patterns and factors associated with survival. The dashboard allows users to explore passenger demographics, survival outcomes, ticket fares, and passenger class through interactive visualizations and filters.

## Objectives

The goal of this project was to:

- Analyze overall passenger survival rates.
- Compare survival rates by gender and passenger class.
- Explore relationships between age, fare, and survival.
- Build an interactive dashboard using Power BI.
- Present key findings through clear visualizations.

---

## Business Questions

This dashboard answers questions such as:

- What percentage of passengers survived?
- Which gender had the highest survival rate?
- Which passenger class experienced the highest survival rate?
- How were passengers distributed across the three passenger classes?
- Did higher ticket fares correspond with higher survival rates?
- Which age groups made up the largest portion of passengers?
- How did gender and passenger class interact?

---

## Key Insights

- Overall survival rate was approximately **38%**.
- Female passengers had significantly higher survival rates than male passengers.
- First-class passengers were more likely to survive than second- and third-class passengers.
- Third-class passengers represented the largest share of passengers.
- Higher ticket fares generally corresponded with greater survival likelihood.
- Young adults represented the largest passenger age group.
